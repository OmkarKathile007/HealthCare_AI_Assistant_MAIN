## Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm 9.x or higher

### Installation



1. Clone the repository:
```bash
git clone [repository-url]
cd HealthCare_AI_Assistant_MAIN
cd project
```


3. Install dependencies:
```bash
npm install
```
4. Start the development server:
```bash
npm run dev
```

For Backend
5.Open new Terminal :
```bash
cd backend
node db2.connection.js
```
